Manual Setup (if needed)

If you prefer to set up manually, follow these steps:

1
Open Chrome Extensions
Go to chrome://extensions in Chrome
2
Enable Developer Mode
Toggle "Developer mode" ON (top right corner)
3
Load Harbor Extension
Click "Load unpacked" and select: /Library/Application Support/Harbor/chrome-extension
4
Load Web Agents Extension
Click "Load unpacked" again and select: /Library/Application Support/Harbor/web-agents-chrome
5
Configure Native Messaging
Copy both extension IDs (32 characters each) from Chrome, then run: /Library/Application Support/Harbor/configure-extension-id.sh
6
Restart Chrome
Quit and reopen Chrome for changes to take effect
