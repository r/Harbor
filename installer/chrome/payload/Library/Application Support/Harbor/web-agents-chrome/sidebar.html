Text Generation
Sites can send prompts to an AI and get responses back. This is the basic building block for any AI-powered feature.

Tool Access
The site's code can call specific tools directly‚Äîe.g., "call the Gmail tool to fetch inbox." The developer decides exactly which tools run and when.

Agentic Tool Use
The AI autonomously decides which tools to call. Give it a task like "summarize my unread emails" and it figures out the steps itself. More powerful, but the AI has more control.
agent.run()
Browser Interaction
The AI can click buttons, fill out forms, and scroll on the current page. Useful for automating tasks within a single site.
agent.browser.click() agent.browser.fill() agent.browser.scroll()
Browser Control
The AI can open new tabs, navigate to URLs, and fetch content from other websites. Enables research and multi-site workflows.
agent.browser.tabs agent.browser.navigate() agent.browser.fetch()
